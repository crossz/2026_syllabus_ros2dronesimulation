Message Types & Interface Definition

Built-in Message Types

  • std_msgs: String, Int32, Float64, Bool, Header
  • sensor_msgs: Image, PointCloud2, Imu, LaserScan
  • nav_msgs: Odometry, Path, OccupancyGrid
  • geometry_msgs: Twist, Pose, Transform, Quaternion

Custom .msg Files

  • Definition: Field type and name pairs
  • Example:
  • string name
  • int32[] data
  • geometry_msgs/Pose pose

IDL Interface Definition

  • IDL: Interface Definition Language
  • ROS2: Uses OMG IDL standard
  • Conversion: .msg → .idl → C++/Python
  • Tools: rosidl_generate_interfaces

Package Management

  • Package: Unit of software distribution
  • CMakeLists: Build configuration
  • package.xml: Metadata and dependencies
  • msg/: Custom message directory
