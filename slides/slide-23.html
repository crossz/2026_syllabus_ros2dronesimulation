Gazebo Evolution History

From Classic to Harmonic

Classic

2002-2022

The original Gazebo simulator. 20 years of development, ROS integration pioneer.

Ignition

2018-2022

New architecture. Modular design. Modern C++ core. Codenamed "Fortress".

Harmonic

2022-Now

Current LTS version. Long-term support. gz-sim API. Fusion of Classic+Ignition.

Ubuntu Version Binding

● Gazebo Classic → Ubuntu 16.04/18.04/20.04
● Gazebo Ignition → Ubuntu 20.04/22.04
● Gazebo Harmonic → Ubuntu 22.04+ (Recommended)

Physics Engines Supported

ODE: Open Dynamics Engine | Bullet: Real-time physics | Simbody: Multibody dynamics | DART: Dynamic Animation and Robotics Toolkit
