18 、 ROS2 Rqt 工具箱  
本节教程的文件夹中有演示动画 git ，可以直观看到本节课程的案例的操作流程
1 、Rqt 简介  
Rqt 是 ROS 提供的另外一种模块化可视化工具，正如 Rqt 的命名，它和 Rviz 一样，也是基于 QT 可视化工具
开发而来，在使用前，我们需要通过这样一句指令进行安装，然后就可以通过 rqt 这个命令启动使用了。
 
2 、安装  
一般只要安装的是 desktop 版本就会默认安装 rqt 工具箱；
如果安装 ros 时不是安装的完整版本需要安装可以以如下方式安装
 
3 、启动  
常用的rqt 启动命令有：
方式 1 ：rqtsudo apt install ros-${ROS_DISTRO}-rqt*

方式 2 ：ros2 run rqt_gui rqt_gui
 
4 、插件使用  
启动 rqt 之后，可以通过 plugins 添加所需的插件：

在plugins 中包含了话题、服务、动作、参数、日志等等相关的插件，我们可以按需选用，方便的实现
ROS2 程序调试。使用示例如下。
 
4.1 、 topic 插件  
添加 topic 插件并发送速度指令控制乌龟运动。
 
4.2 、 service 插件  
添加  service 插件并发送请求，在制定位置生成一只乌龟。

 
4.3 、参数插件  
通过参数插件动态修改乌龟窗体背景颜色。
